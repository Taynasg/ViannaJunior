Dentre as TAGS importantes de um site podemos citar:

  • HTML
  • HEAD
  • BODY

No HTML 5 temos uma estrutura básica que pode ser identada assim:

  1. HTML
    1. HEAD
      1. META
      2. TITLE
    2. /HEADE
    3. BODY
      1. HEADER
      2. ARTICLE
        1. SECTION
      3. FOOTER
    4. /BODY
  2. /HTML
